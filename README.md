# 🎴 Expanding Cards

Интерактивный UI-проект с карточками, которые расширяются при клике.  
Реализован на HTML, CSS и JavaScript.

---

## 🚀 Демо

👉 https://dias-frontend.github.io/expanding-cards/

---

## 📌 Функционал

- Анимация расширения карточек  
- Клик → активное состояние  
- Плавные переходы  
- Фоновые изображения  
- Появление текста  
- Адаптивный UI  

---

## 🛠 Технологии

HTML5  
CSS3 (Flexbox, Transitions)  
JavaScript (DOM, Events)

---

## 📂 Структура проекта

project/  
├── index.html  
├── styles.css  
└── app.js  

---

## 🖥️ Как запустить

Склонируй проект:

git clone https://github.com/dias-frontend/expanding-cards.git

Открой файл:

index.html

Готово 🎉

---

## 🎯 Как пользоваться

- Кликни по карточке  
- Карточка увеличится  
- Появится текст  
- Можно переключаться  

---

## 🧠 Как работает

- Все карточки имеют класс `.slide`  
- Активной даётся `.active`  
- При клике обновляются классы
