# 🎴 Expanding Cards — Интерактивные карточки (Frontend)

Интерактивный UI-проект с карточками, которые расширяются при клике.  
Реализован на **HTML, CSS и JavaScript без библиотек**.

---

## 🚀 Возможности

✅ Анимация расширения карточек  
✅ Клик по карточке → активное состояние  
✅ Плавные переходы (CSS transitions)  
✅ Фоновое изображение для каждой карточки  
✅ Текст появляется при активации  
✅ Адаптивный UI 📱

## 🧠 Логика работы

### 📌 Основной принцип

- Все карточки имеют класс `.slide`  
- Активная карточка получает класс `.active`  
- При клике:
  - удаляется `.active` у всех карточек  
  - добавляется только выбранной  

---

### 💻 Пример логики

```js
slides.forEach(slide => {
  slide.classList.remove('active')
})
